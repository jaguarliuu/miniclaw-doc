# 🏠 课程简介

> **技术无废话，内容有灵魂。**

欢迎来到 **MiniClaw AI Agent 实战课程**——一门从零构建生产级 AI Agent 框架的深度课程。

## 这是一门什么样的课程？

这不是普通的 AI 应用开发课。

市面上大多数 AI 课程停留在"调用 API 做问答"的层面。学员学完后只会用现成框架（LangChain、AutoGPT）堆砌功能，**却不理解框架背后的设计原理**。

MiniClaw 的核心差异化：

- ❌ **不使用任何 AI Agent 框架**（LangChain、LlamaIndex、Semantic Kernel 等）
- ✅ **从零手写完整 Agent 架构**，深刻理解每一行代码的设计动机
- ✅ **基于 OpenClaw 工业级实践**，对标真实生产环境
- ✅ **让 AI 脱离问答场景**，通过工具调用真正影响现实世界

## 你将学到什么？

### 核心能力

| 能力 | 说明 |
|------|------|
| **五层架构设计** | Control / Execution / Extension / State / Automation |
| **ReAct 循环** | Think → Act → Observe 的实现原理 |
| **工具系统** | 参数校验、权限控制、HITL 确认机制 |
| **Memory 工程** | Markdown 真相源、向量检索、Compaction |
| **Skills 系统** | 解析、选择、工具白名单 |
| **SubAgent** | 并行派生、屏障等待、结果汇总 |

### 实战产出

完成课程后，你将获得：

- ✅ 完整的 AI Agent 后端服务（Spring Boot）
- ✅ Web UI 可视化界面（Vue 3）
- ✅ CLI 命令行客户端（Java）
- ✅ Docker Compose 一键部署方案

## 适合谁学习？

| 学员类型 | 前置要求 | 学习收益 |
|---------|---------|---------|
| **后端开发工程师** | Java/Spring Boot 基础 | 转型 AI 应用开发，掌握 Agent 架构设计 |
| **AI 应用开发者** | 会用 LangChain 等框架 | 理解框架原理，具备定制化能力 |
| **架构师/技术 Leader** | 系统设计经验 | 为团队设计 AI Agent 系统架构 |
| **计算机专业学生** | 数据结构 + 网络编程 | 实战项目经验，提升就业竞争力 |

## 技术栈

| 类别 | 技术 |
|------|------|
| **后端** | Java 24 + Spring Boot 3.4 + Spring WebFlux |
| **数据库** | PostgreSQL 16 + pgvector |
| **前端** | Vue 3 + TypeScript + Vite |
| **部署** | Docker Compose |
| **LLM** | OpenAI 兼容 API（DeepSeek / 通义千问 / Ollama） |

## 课程结构

本课程共 **18 章 + 3 附录**，预估学习时长 **36-42 小时**。

- **第 1-2 章**：AI Agent 全景 + 实操体验（先用后学）
- **第 3-6 章**：基础底座 + LLM + 网关 + 前端
- **第 7-10 章**：核心系统（工具、ReAct、Skills、Memory）⚠️ 重点
- **第 11-14 章**：高级功能（SubAgent、Cron、MCP、Datasource）
- **第 15-17 章**：实战场景 + 进阶
- **第 18 章**：课程总结
