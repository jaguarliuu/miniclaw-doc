# 第0章 - 课程导学

:::info 预估时长
10 分钟

## 欢迎来到 MiniClaw

你好，欢迎来到 **MiniClaw AI Agent 实战课程**！

在开始这段学习旅程之前，我想先和你聊聊：这门课是什么、适合谁、你能收获什么。

## 这门课讲什么？

简单说：**从零手写一个 AI Agent 框架**。

不是教你调用 API，不是教你用 LangChain，而是带你一行一行写出一个完整的 AI Agent 系统——包括：

- 🧠 **ReAct 循环**：让 AI "思考-行动-观察"的核心机制
- 🛠️ **工具系统**：让 AI 能读写文件、执行命令
- 📚 **技能系统**：给 AI 装上"专业技能"
- 💾 **记忆系统**：让 AI 记住你的偏好和历史
- ⏰ **自动化**：让 AI 定时主动工作

## 这门课适合你吗？

**适合，如果你：**
- 会 Java / Spring Boot 基础
- 想转型 AI 应用开发
- 不满足于"会用框架"，想理解原理
- 希望具备设计 AI 系统的能力

**可能不适合，如果你：**
- 完全不会编程
- 只想做 Prompt Engineering
- 只想快速出个 Demo

## 学习建议

1. **先用后学**：第 1-2 章先体验 AI Agent 的能力，建立直观感受
2. **动手实践**：每章都有代码，跟着敲一遍比看十遍有用
3. **理解"为什么"**：不只是"怎么做"，更重要的是设计动机
4. **用 AI 辅助学习**：让 AI 帮你解释概念、review 代码
