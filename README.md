# MiniClaw AI Agent 实战课程文档站

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm start

# 构建生产版本
npm run build
```

## 结构

```
miniclaw-docs/
├── docs/                    # 课程内容
│   ├── intro.md            # 课程简介
│   ├── chapters/           # 各章节
│   └── appendix/           # 附录
├── src/
│   ├── components/         # React 组件
│   ├── css/                # 样式文件
│   └── pages/              # 页面
├── static/                  # 静态资源
├── docusaurus.config.js    # Docusaurus 配置
└── sidebars.js             # 侧边栏配置
```
